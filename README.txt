The Folder included with this README file is the open-source COMBIgor package for commercial Igor Pro. Included within is an installer (COMBIgor_Installer.pxp) that should place the package folder in the right location to enable full functionality.

After installing COMBIgor, please take time to familiarize yourself with its functionality by following the tutorial available through COMBIgor. Also you can take the Igor Pro tutorial if you need to familarize youself with this softweare package.

More information about COMBIgor installation and use is available at www.COMBIgor.com
